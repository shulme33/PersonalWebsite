### Starting the Server

In the project directory, you can run:

`npm install`

Followed by:

`node app.js`

This will start the app server on port 5000;

### Starting the client

To start the client, run the following:

`cd client`

`npm install`

`npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
